# 街デザイン仕様

Chill Night Town の各街（世界）のコンセプト・デザイン方針をまとめる。値域・制約・処理フローの正は要件定義書 v2／ユースケース設計書 v2／テーブル定義書 v2 にあり、本フォルダは**世界観と見た目の設計意図**を記録する。

| :--- | :--- | :--- | :--- |
| `starHill` | 星見の丘 | 空（星空）そのものが育つ、見上げる場所 | [星見の丘](星見の丘.md) |

## 全街共通の設計

- **空色 = （季節 × 時刻）→ 空色**：季節で背景画像は変えないが、季節ごとに時間帯の空色が変わる（夏は 19:00 頃まで夕方、冬は 17:00 には暗い）。オーバーレイの仕組みは全街共通で、街ごとに色パレットだけ変える。
- **成長は 2 軸**：レベル軸（模様・地上＝上がったら下がらない）× 時間軸（空色＝その夜の時刻）。
- **視点は街ごとに変える**（同じアプリでも"場所を移動した"感を出す）:

| 街 | 視点 | 視線 | 主役 |
| :--- | :--- | :--- | :--- |
| starHill | 空を見上げる（あおり） | 下→上 | 空 |
| nightTown | 街を見下ろす（俯瞰） | 上→下 | 街の灯り一望 |
| castleTown | 通りに立って見渡す（水平〜ややあおり） | 水平 | 街並みと夜桜 |
| snowTown | 窓辺から雪景色を望む（囲われた水平）※検討中 | 水平／内→外 | 降る雪と灯り |

- **各街に Lv5 到達演出／特別な空**（ご褒美の特別な瞬間）を持たせる。starHill＝星が弧を描く／castleTown＝花びらが舞う／nightTown＝流れ星が走る／snowTown＝オーロラがかかる。
- **黒猫**（nightTown で展望テラスに座る相棒）を全街共通のモチーフにするかは検討中（[夜の街](夜の街.md) 参照）。
