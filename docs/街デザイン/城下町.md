# 街デザイン仕様 — 夜桜の城下町（castleTown）

Chill Night Town の街のひとつ。星見の丘（`starHill`）が「空が育つ特別な場所」だったのに対し、城下町は**"人の営みの灯りが増える"王道の街**であり、他の街の基準になる存在として設計する。

- **表示名**: 夜桜の城下町 ※仮。「桜灯りの城下町」なども候補
- **街コード / ディレクトリ名**: `castleTown`
- **コンセプトの毛色**: 和風。夜桜と提灯・軒先の暖色の灯りが街を照らす。急かさない世界観に"和の静けさ"を合わせる。

---

## 1. コンセプト

**「夜桜の下、灯りがひとつずつともる城下町」**

石畳、提灯、格子戸の暖色の光。天守を遠景に、夜桜が街を淡く照らす。星見の丘が「空（上）」を見上げる場所なら、城下町は**街並み（地上）**が主役。人の営みの灯りが少しずつ増えていく、王道の"帰ってくる夜の街"。

### 象徴イメージ

**満開の夜桜と、点々とともる提灯・軒先の灯り**。視線を街並みへ向ける、地に足のついた温かさ。

### トーン（コンセプト遵守）

- 賑わいは描くが、急かす・競わせる方向には振らない。あくまで**静かに満ちていく**灯り。
- 和の情緒（石畳・のれん・橋・天守）で"落ち着いて過ごせる夜"を表現する。

---

## 2. 視点（構図）

**通りに立って、天守と夜桜を見渡す**（水平〜ややあおり・中景）。

- プレイヤーが**街の中を歩いている**視点。石畳の通りの先に天守、手前に夜桜。
- **夜桜を手前（近景）に配置**し、奥に街並みを見せることで奥行きを出す。他の街にない「奥行きのある近さ」がこの街の個性。

### 全街の視点の住み分け（対比のための一覧）

| 街 | 視点 | 視線 | 主役 | 印象 |
| :--- | :--- | :--- | :--- | :--- |
| starHill | 丘の上で空を見上げる（あおり） | 下→上 | 空 | 開放・静寂・仰ぐ |
| nightTown | 丘の上から街を見下ろす（俯瞰） | 上→下 | 街の灯り一望 | 眺める・遠い |
| **castleTown** | **通りに立って見渡す／天守を仰ぐ（水平〜ややあおり）** | **水平** | **街並みと夜桜** | **街の中にいる・巡る** |
| snowTown | 窓辺（室内）から雪景色を望む（囲われた水平）※検討中 | 水平／内→外 | 降る雪と灯り | 籠る・あたたかい |

---

## 3. 空の設計 — 2 軸の掛け算

空色の仕組みは全街共通（星見の丘と同じ）。**城下町は色パレットを"和"に寄せる**（夕方の茜色、深夜の藍鼠など）と統一感が出る。

| 軸 | 対応するもの | 変化のしかた |
| :--- | :--- | :--- |
| レベル軸 = 灯りと賑わい＋桜 | 提灯・軒先の灯りの数、桜の咲き具合、建物 | 上がったら戻らない |
| 時間軸 = 空の色 | 空のグラデーション色 | **（季節 × 時刻）**でなめらかに変わる |

### 季節 × 時刻による空色（全街共通の仕組み）

季節で背景画像は変えないが、**季節によって時間帯ごとの空色が変わる**。例: 夏は 19:00 頃まで夕方の空、冬は 17:00 には暗くなる。空色オーバーレイの入力に「季節」を 1 つ加える形で扱い、これは全街に共通して効く。

---

## 4. レベル構成（Lv1〜5）

街並み（地上）と桜が同時に育つ。街は下がらないため、各 Lv は前段を含んで積み上がる。

| Lv | 街並み（地上） | 桜 |
| :--- | :--- | :--- |
| **Lv1** | 小さな一軒家＋提灯ひとつ、石畳の一角 | つぼみ〜数輪 |
| **Lv2** | 家が数軒、軒先の灯りが増える | 三分咲き |
| **Lv3** | 通りらしくなる（茶屋・橋・のれん）、遠くに天守が見え始める | 五分咲き |
| **Lv4** | 賑わう町並み、提灯の列、天守がはっきり | 満開に近い |
| **Lv5** | 天守を望む完成した城下町、灯りが街いっぱい | **満開の夜桜＋花びらが舞う** |

### Lv5 の到達演出（星見の丘の"軌跡"に相当）

星見の丘が「星が弧を描く」なら、城下町は**「夜桜の花びらが一斉に舞う／灯りが街いっぱいにともる」**瞬間。地上の街ならではのご褒美とする。

---

## 5. レイヤー構成（実装メモ）

奥 → 手前の順。レベルと時間を独立に差し替えられるように分ける。

1. `空（時間帯の色オーバーレイ）` — （季節 × 時刻）で補間したグラデーション
2. `遠景（天守・街並みのシルエット、Lv 別）`
3. `中景（通り・提灯・軒先の灯り、Lv 別）`
4. `近景（手前の夜桜、Lv 別／Lv5 で花びらが舞う）`
