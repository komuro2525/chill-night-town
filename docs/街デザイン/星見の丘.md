# 街デザイン仕様 — 星見の丘（starHill）

Chill Night Town の街（世界）のひとつ。既存の 3 つ（城下町 `castleTown` / 夜の街 `nightTown` / 雪国 `snowTown`）とは成長の見せ方が異なるため、その設計意図をここに残す。

- **表示名**: 星見の丘
- **街コード / ディレクトリ名**: `starHill`
- **命名の意図**: 既存が `〜Town` 揃いのところ、あえて `starHill` とすることで「ここだけ"街"ではない特別な場所」を型でも示す。「星を見る」という**行為**を名前にし、"見上げる側に立つ"というこの場所の本質を表す。

---

## 1. コンセプト

**「街の灯りではなく、星空そのものが育っていく夜」**

他の街が"人の営みの灯り（窓の数）"で夜を満たしていくのに対し、星見の丘は光の少ない高原で**空を見上げる側**に立つ。地上は静かなまま、夜が更けるほど空が主役になっていく。Chill Night Town の核である「夜」を最も素直に体現する場所。

### 象徴イメージ

**星がくるくると弧を描く星景（星の軌跡）**。長い時間そこに居たからこそ現れる光の軌跡は、**「この場所に何度も帰ってきた積み重ね」**の視覚的メタファーそのもの。ゆえに軌跡は Lv5＝完成の到達点に置く。

### トーン（コンセプト遵守）

- 建物を増やして画面を埋めるのではなく、**空が深まる・環境が整う**ことで成長を見せる。急かさない・競わせない思想を、成長曲線そのもので体現する。
- NPC・完成メッセージも「灯りが増えた」ではなく、**「今夜も星を見に来てくれた」**という、見上げる場所ならではの温度で綴る。

---

## 2. 空の設計 — 2 軸の掛け算

空は独立した 2 つの軸で構成する。**積み重ね（模様）× 今夜（色）**で一枚の空になる。

| 軸 | 対応するもの | 変化のしかた |
| :--- | :--- | :--- |
| **レベル軸 = 空の"模様"と地上** | 星の濃さ・天の川・建物 | 上がったら戻らない（＝積み重ねの記録） |
| **時間軸 = 空の"色"** | 空のグラデーション色 | **（季節 × 時刻）**でなめらかに変わる |

模様（星・天の川・軌跡）は据え置き、その上に**時刻で補間した空色を薄く重ねる**。この分離により、星の軌跡のような"長時間露光の一枚絵"と「時刻で空が変わる」演出が競合しない。

### 季節 × 時刻による空色（全街共通の仕組み）

季節で背景画像は変えないが、**季節によって時間帯ごとの空色が変わる**。例: 夏は 19:00 頃まで夕方の空、冬は 17:00 には暗くなる。空色オーバーレイの入力に「季節」を 1 つ加える形で扱い、これは全街に共通して効く。

### 時間軸（空の色）のイメージ

| 時刻 | 空の色 | 星の見え方 |
| :--- | :--- | :--- |
| 18:00 頃 | 地平線に薄いオレンジ／紫、上は藍 | まだ淡い |
| 21:00 頃 | 全体が濃紺〜黒に近い | くっきり見え始める |
| 24:00 頃 | 最も深い夜の色（青みの濃い黒） | 天の川・星が主役 |
| 4:00〜5:00 | 地平線がわずかに白む | 星が引いていく |

- 色は画像ではなく**グラデーションのオーバーレイ**で持つ（軽く・滑らか・時刻補間しやすい）。基準時刻の色を定義し、間を補間する。
- 時刻と時間帯の対応は学習日ロジック（`study-day`）と揃え、破綻を防ぐ。
- 基準時刻の色は定数化し、他の街でも同じ仕組みを流用できるようにする（街ごとに色パレットだけ差し替え）。

## 3. レベル構成（Lv1〜5）

地上（建物）と空（模様）が同時に育つ。街は下がらないため、各 Lv は前段を含んで積み上がる。

| Lv | 地上（建物） | 空（模様） |
| :--- | :--- | :--- |
| **Lv1** | 望遠鏡で空を見る人＋焚火 | 星まばら |
| **Lv2** | テント＋望遠鏡＋ランタンの灯り | 星が増える |
| **Lv3** | 望遠鏡小屋 | 天の川がうっすら |
| **Lv4** | ドーム型天文台＋あずまや | 天の川くっきり・流れ星 |
| **Lv5** | ドーム＋草花・小径（訪れたくなる名所へ） | 満天、**到達演出で星が弧を描く（星の軌跡）** |

設計意図:

- **Lv3→Lv4 は「小屋→ドーム」で明確な格上げ**。ここが視覚的な転換点。
- **Lv5 は建物を大きくするのではなく"環境が整う"**（草花・小径）。「人が訪れる名所になった」を静かに見せ、コンセプトを保ったまま完成感を出す。
- 空は**地上より模様の変化で完成へ向かう**（Lv4 で天の川、Lv5 で満天）。「空が主役」を最後まで貫く。

### 星の軌跡（Lv5）の扱い

星がくるくる回る軌跡はインパクトが大きい一方、"長時間露光の一枚絵"なので常時背景には向かない。よって:

- 通常の Lv5 背景は**濃い天の川＋満天**とする。
- **星の軌跡は Lv5 到達の演出でのみ表示**（到達の瞬間だけ空が回る＝ご褒美）。「長く通った夜が、空に軌跡を残した」という体験に絞る。

## 4. レイヤー構成（実装メモ）

奥 → 手前の順。レベルと時間を独立に差し替えられるように分ける。

1. `星の模様（Lv 別）` — 星・天の川・（Lv5 演出時のみ軌跡）
2. `時間帯の色オーバーレイ` — 時刻補間したグラデーション
3. `地上の建物（Lv 別）` — 焚火／テント／小屋／ドーム
4. `手前の草花・光源（Lv 別）` — ランタン・草花・小径
