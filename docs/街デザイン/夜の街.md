# 街デザイン仕様 — 夜の街（nightTown）

Chill Night Town の街のひとつ。**すでに Lv1〜5 の背景画像が実装済み**（`assets/images/home/nightTown/lv1〜5.png`）。本仕様は既存画像の実態に合わせて記述する。

- **表示名**: 夜の街 ※仮
- **街コード / ディレクトリ名**: `nightTown`
- **コンセプトの毛色**: 海辺の夜。高台の展望テラスから、猫と一緒に海辺の街と海を眺める。

---

## 1. コンセプト

**「猫と眺める、海辺の夜の街」**

高台の展望テラスに黒猫が一匹座り、眼下に広がる海辺の街と、月明かりを映す海を静かに眺めている。灯りが少しずつ増えていく街を、少し離れた特等席から見守る——「またこの世界に帰ってきたい夜」を素直に体現する、シリーズの顔とも言える街。

### 象徴イメージ

**展望テラスに座る黒猫と、月明かりを映す海／街の灯り**。黒猫は全レベル共通で同じ場所に座り、プレイヤーの相棒・分身のように夜を眺め続ける。遠景に灯台・三日月・天の川。

### トーン（コンセプト遵守）

- 街の賑わいは"眼下の眺め"として置くため、うるさくならず**静かに満ちていく**灯りになる。
- 猫の後ろ姿が「急かさず、ただ隣で夜を過ごす」というアプリの温度をそのまま表す。

---

## 2. 視点（構図）

**高台の展望テラスから海辺の街と海を眺める**（やや見下ろし＋眺望）。

- 手前（近景）: 石畳の展望テラス、鉄柵、街灯、道標、そして座る黒猫。
- 中景: 斜面に広がる海辺の街並み（家々・教会・時計塔）。
- 遠景: 月明かりを映す海、対岸の灯台、三日月、星空・天の川。
- 星見の丘＝見上げ、城下町＝街の中の水平、雪国＝窓辺、に対し、夜の街は**"高台からの眺望"**で対比する。

---

## 3. 空の設計 — 2 軸の掛け算

空色の仕組みは全街共通（[README](README.md) 参照）。夜の街は深い群青（月明かりの海に映える青夜）に寄せる。

| 軸 | 対応するもの | 変化のしかた |
| :--- | :--- | :--- |
| レベル軸 = 街の灯りと賑わい | 街灯の点灯・窓の灯り・建物の数 | 上がったら戻らない |
| 時間軸 = 空の色 | 空のグラデーション色 | **（季節 × 時刻）**でなめらかに変わる |

- **注意**: 既存画像には星空・三日月・天の川が描き込まれている。時間色オーバーレイを重ねる際は、**画像の空と色が喧嘩しない**基準色にする必要がある。
- **月と天気の兼ね合い**: 要件 3.4 で月・満月は「夜の天気」が所有する要素。背景画像に三日月が固定で描かれているため、天気演出（月・満月・闇夜など）との重なりをどう扱うかは要確認（下記 6 章）。

---

## 4. レベル構成（Lv1〜5）※既存画像の実態

眼下の街の灯りが増え、空も少しずつ賑やかになる。街は下がらないため、各 Lv は前段を含んで積み上がる。

| Lv | 眼下の街と空（実装済み画像の傾向） |
| :--- | :--- |
| **Lv1** | 数軒＋教会のみ。街灯は灯っておらず、全体に暗く静か。海と三日月 |
| **Lv2** | 家と窓の灯りが増え始める |
| **Lv3** | 街灯が灯り（暖色）、窓の灯りが増え、時計塔が現れる。天の川がはっきり |
| **Lv4** | 街並みがさらに賑わい、灯りの密度が上がる |
| **Lv5** | 街いっぱいの灯り、時計塔・自転車・ベンチなどが揃い、**空に流れ星が走る** |

### Lv5 の到達演出

星見の丘＝星が弧を描く、城下町＝花びらが舞う。夜の街は**「流れ星が走る／街灯と窓の灯りが一斉にともる」**瞬間（既存 Lv5 画像の流れ星に準拠）。

## 5. レイヤー構成（実装メモ）

現状は Lv ごとに**一枚絵**で実装済み。今後レイヤー分け・時間色オーバーレイを載せる場合の想定は以下。

1. `空（時間帯の色オーバーレイ）` — （季節 × 時刻）で補間したグラデーション（既存画像の星空と両立させる）
2. `遠景（海・灯台・街並みのシルエット）`
3. `中景（街の灯り・建物、Lv 別）`
4. `近景（展望テラス・鉄柵・街灯・黒猫）`

## 6. 未確定事項（実装前に確定させる）

- [ ] **表示名の確定**（実態は「海辺の夜の街」。表示名を実態に寄せるか要検討）。
- [ ] **時間色オーバーレイと既存一枚絵の両立**: 既存画像は空・海・街が 1 枚に焼き込まれている。時刻で空色を変える仕組みを後載せするか、この街は一枚絵のまま特例扱いにするかを決める。
- [ ] **月・天気との兼ね合い**: 背景の三日月と、天気演出（月／満月／闇夜など・要件 3.4）の重なりの扱い。
- [ ] **黒猫を他の街にも共通モチーフとして置くか**（相棒キャラとして全街に登場させると世界観が繋がる）。
- [ ] **必要素材の洗い出し**: 既存画像の差し替え・レイヤー化の要否を [必要素材一覧](../必要素材一覧.md) に反映する。
