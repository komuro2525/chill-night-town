# 街デザイン仕様 — 雪国（snowTown）

Chill Night Town の街のひとつ。他の 3 街が「外に立って（または見上げて）夜を眺める」のに対し、雪国は**窓辺の内側から雪景色を望む**、囲われたあたたかい視点をとる。

- **表示名**: 雪国 ※仮
- **街コード / ディレクトリ名**: `snowTown`
- **コンセプトの毛色**: 冬・雪。冷たい外気と、こちら側の暖色の灯り。「また帰ってきたい夜」を最も体現する街。

---

## 1. コンセプト

**「窓の内側から眺める、雪の降る夜」**

外は静かに雪が降り、窓の向こうに雪明かりの街。手前（こちら側）には暖色の室内の灯り。**冷たい外と、あたたかい内側**の対比が、アプリのコンセプト「また帰ってきたい夜の街」をまっすぐに表現する。

### 象徴イメージ

**降る雪と、窓越しの雪明かり／街の灯り**。窓枠がフレームになり、"守られた居場所から夜を眺める"感覚を作る。

### トーン（コンセプト遵守）

- 寒さは"外の景色"として置き、こちら側は常にあたたかい。急かさない・責めないの思想を、"帰る場所の温度"で表現する。
- 静けさ（降雪の無音感）を大切にする。

---

## 2. 視点（構図）

**窓辺（室内）から雪景色を望む**（囲われた水平・近景）。

- 視線は内→外。手前に窓枠・室内の暖色光（近景）、窓の外に雪と街の灯り。
- 星見の丘＝見上げ、夜の街＝見下ろし、城下町＝街の中の水平、に対して、雪国は**"囲われた水平"**で対比する。4 街で最も"近く・守られた"視点。

> ※ 屋外に立つ構図（雪原に立って街や空を望む）も候補。窓辺案はコンセプト表現力が高いぶん、他画面（ホーム UI）との相性を実機で確認したうえで確定する。

---

## 3. 空の設計 — 2 軸の掛け算

空色の仕組みは全街共通（[README](README.md) 参照）。雪国は冷色寄り（青みの強い夜、雪明かりで地表が明るい）に寄せる。

| 軸 | 対応するもの | 変化のしかた |
| :--- | :--- | :--- |
| レベル軸 = 雪の街の灯りと積雪 | 窓外の灯りの数・積雪・建物 | 上がったら戻らない |
| 時間軸 = 空の色 | 空のグラデーション色 | **（季節 × 時刻）**でなめらかに変わる |

- 雪は**雪明かりで夜でも地表が明るい**のが特徴。空色オーバーレイに加え、地表（雪面）の明度も時刻で調整すると冬夜らしさが出る。
- 窓辺構図では空の面積が小さいので、**窓の外の雪と灯り**を主役にする。

---

## 4. レベル構成（Lv1〜5）

窓の外の雪の街が育っていく。街は下がらないため、各 Lv は前段を含んで積み上がる。

| Lv | 窓の外（雪の街） |
| :--- | :--- |
| **Lv1** | 雪原に一軒、灯りひとつ。降雪はまばら |
| **Lv2** | 家が数軒、窓の灯りが増える |
| **Lv3** | 雪の通り・並木、街らしくなる |
| **Lv4** | 賑わう雪の街、灯りが増え積雪も豊かに |
| **Lv5** | **雪明かりに満ちた街。窓の外いっぱいに灯りと雪、空にオーロラ** |

### Lv5 の空 — オーロラ

雪国の最大レベルは、空に**オーロラ**が揺れる。星見の丘の「星の軌跡」に相当する、この街だけの特別な空。冬夜・雪明かりの世界観に自然に馴染み、「ここまで通った夜へのご褒美」としての格を持つ。

- 通常の Lv1〜4 はオーロラなし。**Lv5 でのみ**空にオーロラが現れる。
- オーロラは緩やかに揺らぐ演出（急がず・静かに）にとどめ、うるさくしない。

### Lv5 の到達演出

星見の丘＝星が弧を描く、城下町＝花びらが舞う、夜の街＝流れ星が走る。雪国は**「粉雪が舞い、空にオーロラがかかり、窓の外の街が雪明かりで一斉に輝く」**瞬間。囲われた居場所ならではの、静かであたたかいご褒美とする。

---

## 5. レイヤー構成（実装メモ）

奥 → 手前の順。

1. `空（時間帯の色オーバーレイ）` — （季節 × 時刻）で補間したグラデーション
2. `遠景（雪の街・窓外の灯り、Lv 別）`
3. `降雪（手前で舞う雪の演出）`
4. `近景（窓枠・室内の暖色光）` ※窓辺案を採る場合
