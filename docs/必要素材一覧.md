# 必要素材一覧（Chill Night Town）

現在のコードで**暫定（コード描画・絵文字・テンプレート・仮素材）のまま**になっている箇所と、素材が用意されたときの**差し替え先・手順**をまとめる。素材が届いたら、このファイルの該当項目を消し込みながら差し替える。

凡例:

- **状態**: 🔴未制作 ／ 🟡仮素材あり（差し替え前提） ／ 🟢暫定実装（コード描画・絵文字）
- 配置先パスは現状のディレクトリに合わせている。新設が要るものは「（新設）」と記す。

## 2. 音源素材（Phase 7 で再生実装済み）

Phase 7 で再生の仕組みは実装済み。差し替え先は音源の静的マップ `src/constants/audioAssets.ts`
（BGM・環境音は `ambient_sound.code` をキー、効果音・鐘は用途名をキー）。未登録・未制作の音は
「鳴らす経路はあるが無音」で通るため、素材が届いたらこのマップへ足すだけで鳴る。

| 素材                       | 状態      | 内容・配置                                                                                                                           |
| :------------------------- | :-------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| BGM（Lo-Fi 等）            | 🟢2曲あり | `assets/audio/bgm/`。シャッフル再生・ミニプレイヤー実装済み。曲の追加可。クレジット（artist）は `ambient_sound` マスタに記録        |
| 環境音（雨/波/森/夜風 等） | 🟡仮あり  | 要件9。**夜の天気**に応じて自動再生（実装済み）。対応は `src/lib/ambient-select.ts`。現状は仮の rain/wind の2つのみで、雨/嵐→雨音・霧→夜風の暫定対応。天気11種ぶんの素材を調達したら ambient-select.ts に追記 |
| 鐘の音（終了演出）         | 🟢あり    | `assets/audio/ambient/The sound of the bell.mp3`。要件3.3。終了時に再生・BGM/環境音をダッキング（実装済み）                          |
| 目標到達の効果音           | 🟡仮あり  | `assets/audio/ambient/test_目標達成.mp3`（**仮**）。要件5.1。休憩提案の表示時に再生（実装済み）。最終素材へ差し替え                  |
| ポモドーロのフェーズ切替音 | 🔴未制作  | 要件3.1。作業⇄休憩の切り替わりで鳴らす経路は実装済み（`pomodoro_phase`・現状無音）。控えめな音を調達して audioAssets へ登録          |
| UI操作音（効果音）         | 🔴未制作  | 要件9。分類（sfx）と鳴らす仕組みはあるが、鳴らす箇所は素材が届いてから決める（`ui_tap`・現状無音）                                   |

- **音の分類（決定済み）**: BGM／環境音／効果音／鐘の音の4分類・各0〜100（設定画面のスライダー実装済み）。マスタ `ambient_sound` の `sound_type` は `bgm` / `ambient` のまま拡張せず、**効果音・鐘はアプリ側の定数マップ（audioAssets.ts）で持つ**方針に確定。環境音の `ambient_sound` 行はデータモデル整備（将来の個別選択）として未追加＝A案（再生には不要）
- 音量0の音は再生処理自体を行わない。バックグラウンド再生は将来拡張

## 3. 差し替え時の共通注意

- **開発用テストボタン**（`__DEV__` 限定）は素材と無関係だが、リリース前に撤去する（`docs/開発用テストボタン.md` の撤去チェックリスト）
- **DBスキーマに関わる素材**（NPC画像パス・街名など）は、CLAUDE.md の方針に従い**先にテーブル定義書・シードSQLを更新してから**コードを変える
- 差し替えたら、このファイルの該当行の状態を更新する（🔴🟡 → 🟢）
